Этот скрипт выполняет копирование содержимого папки-источника (src) в папку назначения (dst). Вот краткое описание его работы:

Скрипт принимает два параметра: src (исходная папка) и dst (папка назначения) в формате src=путь и dst=путь.
Логика работы скрипта состоит из 4 основных шагов:

Проверка совпадения имен папок src и dst
Удаление мусора (временных файлов) из папки dst
Синхронизация структуры папок (создание и удаление директорий)
Синхронизация файлов (копирование новых, обновление изменившихся, удаление отсутствующих)


При выполнении каждого шага будет выводиться результат в форматированном виде с вертикальным выравниванием символов |
Исходная папка (src) никогда не модифицируется, все изменения производятся только в папке назначения (dst).

Скрипт можно запустить командой..
python ./cp_folder.py src=~/name_folder1/ dst=/media/user/disk1/name_folder1/
