get_info_proj
=============
  Принимает параметр proj=path/name_folder для указания пути к анализируемой папке

  Анализирует содержимое папки и выводит
  ======================================
    Количество папок в проекте
    Количество файлов в проекте
    Общее количество строк во всех файлах
    Количество коммитов (или "no", если папки .git нет)
    Объем памяти, занимаемый проектом (в кБ)

  Вывод соответствует указанному формату.
  Программа обрабатывает возможные ошибки, такие как невозможность чтения некоторых файлов или недоступные файлы.

  Для запуска программы используйте следующий синтаксис
  =====================================================
    python ./get_info_proj.py proj=path/name_folder
    python ./get_info_proj.py proj=path/name_folder --version-only
